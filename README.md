# 📊 Dashboard de Vendas — Planos Sócio Torcedor Corinthians

Dashboard de vendas criado no Excel com base nos planos do programa Fiel Corinthians.

## 📁 Estrutura do arquivo

| Aba | Descrição |
|---|---|
| Dashboard | Visão geral com KPIs e gráficos |
| Bases | Base de dados com 200 registros fictícios | 
| Cálculos | Tabelas Dinâmicas para apoio (PLanilha Oculta) | 
| Assets | Paleta de cores, imagens e referência dos planos (PLanilha Oculta) |

## 📌 Planos incluídos

| Plano | Mensal | Anual |
|---|---|---|
| Fiel Digital | R$ 14,90 | — |
| Minha Vida | R$ 24,50 | R$ 250,00 |
| Minha História | R$ 60,00 | R$ 660,00 |
| Meu Amor | R$ 300,00 | R$ 3.600,00 |
| Minha Cadeira | R$ 700,00 | R$ 8.400,00 |

## 🛠️ Como reproduzir

1. Baixe o arquivo `fiel_corinthians_dashboard.xlsx`
2. Abra no Microsoft Excel (recomendado) ou Google Sheets
3. Navegue pela aba **Dashboard** para visualizar os KPIs e gráficos

## ⚠️ Aviso

Os dados utilizados são **fictícios**, gerados exclusivamente para fins de portfólio.
